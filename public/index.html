Cidades mais limpas, um terreno por vez

Plataforma completa para gestão e monitoramento de terrenos urbanos abandonados. Conectando cidadãos e prefeitura para um ambiente urbano mais seguro e limpo.

Recursos Principais

Nossa plataforma oferece um conjunto completo de ferramentas para gerenciar denúncias e promover a colaboração entre cidadãos e poder público.

Múltiplos Papéis

Sistema completo de roles com administradores, funcionários da prefeitura e cidadãos.

Sistema de Tags

Categorização eficiente de denúncias com sistema flexível de tags.

Comentários

Sistema completo de comentários com moderação e notificações.

Stack Tecnológico

Utilizamos tecnologias modernas e robustas para garantir performance, segurança e escalabilidade.

Node.js

Runtime JavaScript de alta performance

  • Versão 18+
  • Event-driven
  • Non-blocking I/O

Express

Framework web rápido e minimalista

  • Versão 4.x
  • Middleware
  • Routing

Sequelize

ORM robusto e completo para Node.js

  • Versão 6.x
  • Multi-database support
  • Association management

MySQL

Banco de dados relacional confiável

  • Versão 8.x
  • ACID compliance
  • High availability
